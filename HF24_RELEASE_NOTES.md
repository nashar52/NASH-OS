# NASH OS HF24 — Unified Enterprise Workspace Framework

HF24 standardizes every operational surface without changing the MySQL schema or removing existing functions.

## Implemented
- Shared enterprise workspace shell for all operations opened through `openOperation`.
- Role-aware breadcrumbs and consistent workspace title hierarchy.
- Visible MySQL live source-of-truth context and tenant context.
- Unified action bar with Workspace Home, Action Ledger, and Close actions.
- Responsive workspace layout and consistent content spacing.
- Existing Employee 360, Executive Command Center, HR, Manager, Employee, SaaS, AI, compliance, payroll, performance, and learning flows remain intact.

## Boundaries preserved
- No MySQL schema migration.
- No seed or reset.
- No autonomous AI approval.
- Human decision ownership retained.
