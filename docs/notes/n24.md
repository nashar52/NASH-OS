# NASH OS Final Gold Master — Hotfix 01

## Purpose
Fixes Employee `Select My Record` returning no rows in the Final Operating Console.

## Root cause
The backend employee search endpoint returned `employees`, while the final action console expected `profiles`.

## Fix
- Backend now returns both `profiles` and `employees` for compatibility.
- Frontend accepts either `data.profiles` or `data.employees`.
- Added `qa:final-gold-hotfix-01`.

## Locks preserved
- No schema migration.
- No database schema touched.
- No direct database CRUD.
- Runtime receipts only for add/edit/delete requests.
- Human final decision required.
- AI autonomous decision blocked.
