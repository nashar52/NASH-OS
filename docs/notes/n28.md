# Implementation — Final Gold Hotfix 03 Document Vault / Evidence Files Lock

Hotfix 03 converts Evidence from text references into downloadable file actions and adds an Employee File / Document Vault.

Acceptance points:
- `Submit Evidence File` supports file upload and creates a receipt.
- `Edit Draft Evidence` supports replacement/update of a draft file receipt.
- `My Employee File` lists employee documents by category and provides Download File actions.
- Employee document categories include CV / Resume, Academic Certificate, Professional Certificate, Experience Letter, Training Certificate, National ID / Iqama, Employment Contract, Medical / Clearance, Work Evidence, Other.
- Download actions create runtime receipts.
- No schema migration and no database schema touch.
