# NASH OS Final Gold Master — Hotfix 03

## Scope
Adds downloadable evidence files and a controlled Employee File / Document Vault without schema changes.

## Operational changes
- Employee can upload/download own CV, certificates, experience letters, training certificates, contracts, IDs, and task evidence.
- HR can open employee document vault and upload HR-controlled documents after selecting an employee.
- Evidence submission is now a file action, not a text-only reference.
- Document downloads are logged as runtime action receipts.
- Raw files are stored in local browser runtime storage for this local demo; MySQL schema remains untouched.

## Locks
- schemaMigrationIncluded=false
- databaseSchemaTouched=false
- directDatabaseCrudBlocked=true
- runtimeCrudReceiptsActive=true
- downloadableEvidenceFilesActive=true
- employeeDocumentVaultActive=true
